Course Learning Outcomes
Fall 2016

  1. Implement medium-sized Java programs of 500-800 lines of code.
  2. Describe, trace programs using, and implement programs using the following intermediate-level programming concepts:
    • arrays
    • file i/o
    • exception handling
    • inheritance and object-oriented design principles
    • built-in data structures and the Collections library
    • recursion
    • linked lists
